## Overview
Web-based platform developed for Sarai Muebles that provides controlled access to a furniture catalog through an administrator-generated password system.

## Problem
The business needed a way to:
- Share its furniture catalog digitally
- Restrict access to authorized clients only
- Avoid making the full catalog publicly accessible

A simple public website was not sufficient due to privacy and business control requirements.

## Solution
A web system with two access levels:
- **Administrator**: generates and manages access passwords
- **User**: accesses the furniture catalog using a valid password

The platform ensures controlled and secure access to product information.

## My Role
Software Engineer

## Key Features
- Administrator panel for password generation
- User login via access password
- Protected furniture catalog
- Session-based access control
- Clean and simple user interface

## Tech Stack
- PHP
- JavaScript
- HTML
- CSS

## Impact
- Enabled secure digital sharing of the furniture catalog
- Improved control over who can access product information
- Reduced manual sharing of catalogs with clients
- Used by a real furniture business
